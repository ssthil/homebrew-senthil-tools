# homebrew-senthil-tools

Homebrew tap for Senthil's AI developer tools.

```bash
brew tap ssthil/senthil-tools
```

## Tools

| Formula | Description | Repo |
|---|---|---|
| `taskbaton` | Pass work state between agentic tool sessions | [ssthil/taskbaton](https://github.com/ssthil/taskbaton) |
| `llmroute` | Native multi-LLM routing proxy CLI | [ssthil/llmroute](https://github.com/ssthil/llmroute) |

## Install

```bash
brew tap ssthil/senthil-tools
brew install taskbaton
brew install llmroute
```

## Upgrade

```bash
brew upgrade taskbaton
brew upgrade llmroute
```
